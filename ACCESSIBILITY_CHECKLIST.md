# Accessibility Checklist

This website is designed to be accessible and easy to use for everyone.

## Key checks

- [x] Semantic HTML structure with a single main landmark.
- [x] Logical heading order from h1 to h2 to h3.
- [x] Skip-to-content link is available and visible on focus.
- [x] All interactive elements are keyboard accessible.
- [x] Focus states are visible on buttons and links.
- [x] Color contrast is strong for text and interactive elements.
- [x] Text is readable on mobile, tablet, and desktop.
- [x] No meaning depends only on color.
- [x] Motion respects prefers-reduced-motion.
- [x] Forms are adult-facing and would have labels if present.
- [x] Error messages are clear if any form inputs are added later.
- [x] Layout works across small and large screens.

## Notes

- Decorative elements are marked as aria-hidden where appropriate.
- The site avoids hidden or automatic tracking.
- The page is static and does not rely on scripts for basic navigation.
